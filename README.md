# Feu furieux - Jeu Java

## Description
Manoir en Feu est un jeu stratégique développé en Java dans lequel le joueur doit s’échapper d’un manoir avant que la chaleur et les flammes n’épuisent entièrement sa résistance.
Le jeu combine gestion de déplacements, propagation dynamique du danger et collecte d’objets pour survivre.

## Fonctionalités principales
Déplacement du joueur dans un manoir structuré

Système de points de vie affecté par la chaleur

Propagation progressive des flammes

Collecte de clés pour ouvrir des portes

Gestion de la visibilité du joueur (champ de vision)

Plusieurs cartes de manoir avec des dispositions différentes



## Extension ajoutées
Barre de points de vie (indicateur visuel)

Indicateur du nombre de clés possédées

Système de potions pour récupérer de la vie

Ajout de nouveaux manoirs personnalisés

Amélioration de la lisibilité des informations de jeu


## Architecture et choix techniques
Langage : Java

Paradigme : Programmation orientée objet

Séparation claire de la logique du jeu et de l’interface

Gestion des états du jeu (joueur, terrain, obstacles)

Lecture de cartes à partir de fichiers texte


## Difficultés rencontrées
Implémentation de la propagation dynamique des flammes

Gestion de l’affichage des informations de jeu

Coordination entre logique métier et interface graphique


## Améliorations possibles
Système de difficulté progressive

Ajout d’ennemis

Niveaux supplémentaires

Tests unitaires (JUnit)

## Lancer le projet
Ouvrir le projet dans IntelliJ IDEA

Lancer la classe principale furfeux.java

## Contexte
Projet académique réalisé dans le cadre du cursus en informatique à l’Université Paris-Saclay, visant à mettre en pratique la programmation orientée objet et la conception de jeux interactifs.
